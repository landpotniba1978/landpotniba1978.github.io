A dependable media player with an elegant-looking interface

CherryPlayer Portable helps you create playlists with your favorite videos and audio files, then play them anytime. You can also add audio and video streams from certain websites to your playlists. Several playlist options are available, such as repeat or shuffle.

Furthermore, you have the option to search for media files on the Internet, as the application will display any relevant results. This way, you do not need to use the browser to search for media files, as you can use the program to do so internally.

Advanced file features and download options

CherryPlayer Portable allows you to view YouTube videos, but it does not stop here, as it gives you the chance to download certain video files. All you have to do is to select the download directory, then choose the preferred file format from those available.
